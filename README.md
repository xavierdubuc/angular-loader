# Installation


Add following line in dependencies of your `bower.json` :

```
"xd-loader": "git://github.com/xavierdubuc/angular-loader",
```

Add 'xd-loader' as a dependency for you app or module that needs it :

```javascript
angular.module('myModule', [
    (...)
    'xd-loader',
    (...)
]);
```
